---
title: 'vf_post_status'
description: 'Valores posibles para el campo vf_post_status'
---

| Código    | Descripción                                  |
|-----------|----------------------------------------------|
| `success` | Se ha realizado el envío a Veri*Factu.       |
| `failure` | No se ha podido hacer el envío a Veri*Factu. |

<Note>Este campo indica si la factura ha sido enviada a Veri*Factu, pero no indica si el registro ha sido aceptado.</Note>
